This project was called Crowd Dj, a democratic playlist creator. I used the Rdio Javascript Api to import a Flash SWFObject which streamed music from the Rdio servers. I then used the Rdio jQuery library to manage playback. I also used UJS to make Ajax calls using rails remote helpers.

I was called in to turn this "spaghetti" Javascript code into a reusable, scalable & robust OO library. It take two sets of data merges them together and then renders the data into a visual represenation using a map!

Postcode Jobs allows people to search for jobs and employee's based on there postcode.
I used Single Table Inheritance (STI) to register Employers and Jobseekers in one table Users. I also used the Geocoder gem to identify longitude and latitude from a postcode.

A reliable and fast way to find staff, jobs site. Here I used a number of different OO principles, polymorphic association (PA) to sign in users as a Applicant or Employer. Inheritance to allow Devise to use it's sign method and then complete the sign in using our own PA method in a single Users controller. I also bound both sign in's into a transaction method. Mocking to create tests that checked whether Users could sign in using the combination of Devise sign and my own PA method implementation.

This was one of my first Ruby OO programs, I managed to get alot of the code to look beautiful and read very close to English. See the control.rb.
